🚀 Enterprise Network multisite design Simulation — Multi-Site (Pune | Mumbai | Delhi)
Designed and implemented a multi-location enterprise network in Cisco Packet Tracer integrating routing, switching, security, and network services.

🔹 Objective:
To build a scalable and secure network across multiple locations using dynamic routing and advanced Layer 2/Layer 3 concepts.

🔹 Network Architecture:
📍 Pune — VLAN segmentation & inter-VLAN routing
📍 Mumbai — High availability using EtherChannel
📍 Delhi — Backup & recovery using TFTP server

🔹 Key Highlights:
✅ Configured OSPF across all routers for dynamic routing between Pune, Mumbai, and Delhi
✅ Implemented VLAN 100 & VLAN 200 with Inter-VLAN routing in Pune LAN
✅ Configured VTP for VLAN management across switches
✅ Enabled DHCP for automatic IP address distribution to end devices
✅ Implemented EtherChannel in Mumbai LAN for link aggregation and redundancy
✅ Configured Port Security to restrict unauthorized device access
✅ Applied privilege level 5 password security on all routers
✅ Deployed TFTP server in Delhi for configuration backup and recovery

🔹 Concept Used:
This setup demonstrates how enterprise networks integrate multiple technologies — OSPF for routing, VLAN segmentation for network isolation, EtherChannel for redundancy, and security mechanisms to protect infrastructure.

🖥 Tools Used: Cisco Packet Tracer
📡 Technologies: OSPF, VLAN, VTP, Inter-VLAN Routing, EtherChannel, DHCP, Port Security, TFTP

💡 Key Learning:
Real-world networks are not built using a single protocol — combining routing, switching, and security is essential for building scalable and resilient infrastructure.
