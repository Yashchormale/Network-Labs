# Enterprise Network Security & Access Control Simulation

Designed and implemented a secure multi-department enterprise network using Cisco Packet Tracer.

## Technologies Used

* Extended ACL
* EIGRP
* DHCP Relay
* VLANs
* Traffic Filtering

## Key Features

* Department-based access control
* HTTP, DNS, and Database traffic filtering
* DHCP Relay configuration
* Secure server access management
* Connectivity validation and troubleshooting

## Tools Used

* Cisco Packet Tracer
* CLI Diagnostics
* Telnet Testing
